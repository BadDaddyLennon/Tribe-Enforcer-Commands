Frequently Asked Questions

Installation & Setup

  • Owner logs in first on all maps to auto-create entries.
  • Verify asset consolidation before inviting members back.

Avoiding Issues

  • If a non-owner creates a tribe then leaves, the file bugs. Use FixTribe to reset it.

Fixing Player Bugs

  • Step 1: Have the player relog.
  • Step 2: Use AddPlayer if they are missing from the database.
  • Step 3: Use FixTribe if the file remains corrupted.
